 PizzaTime 2 - Telegram Bot

## Описание проекта

Telegram бот для заказа пиццы с функцией кастомизации. Пользователи могут выбирать пиццы из готового меню или создавать свои собственные пиццы с индивидуальным набором ингредиентов.

### Основные функции:
- **Меню** - выбор пицц, размеров, теста и дополнительных ингредиентов
- **Кастомная пицца** - создание пиццы с выбором соуса, сыра и начинок
- **Статус заказов** - отслеживание статуса заказа и времени доставки

## Инструкция по запуску

### 1. Установка зависимостей

npm install

### 2. Настройка окружения
Создайте файл .env в корне проекта и добавьте токен бота:

env
BOT_TOKEN=ваш_токен_бота
PORT=3000

### 3. Запуск бота
bash
# Основной запуск
npm start

# Режим разработки (с авто-перезагрузкой)
npm run dev
