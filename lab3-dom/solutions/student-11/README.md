# Rainbow Borders for KAI.ru

Расширение для браузера добавляет радужные границы к элементам на сайте kai.ru.

## Установка

1. Создать папку с файлами расширения
2. Открыть chrome://extensions/
3. Включить режим разработчика
4. Загрузить распакованное расширение
5. Выбрать папку с файлами

## Функции

- Кнопка переключения стилей
- Сохранение состояния
- Радужные границы для изображений и блоков
